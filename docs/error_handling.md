# ­Ъџе сѓесЃЕсЃ╝сЃЈсЃ│сЃЅсЃфсЃ│сѓ░

PlaywrightAPIсЂ»сђЂсѓ╣сѓ»сЃгсѓцсЃћсЃ│сѓ░СИГсЂФуЎ║ућЪсЂЌсЂЪсѓесЃЕсЃ╝сѓњУЕ│у┤░сЂФУеўжї▓сЂЌсђЂухљТъюсЂФтљФсѓЂсѓІсЂЊсЂесЂїсЂДсЂЇсЂЙсЂЎсђѓсЂЊсѓїсЂФсѓѕсѓісђЂсЂЕсЂ«сѓ╗сЃгсѓ»сѓ┐сЂїтЋЈжАїсѓњт╝ЋсЂЇУхисЂЊсЂЌсЂЪсЂ«сЂІсѓњуЅ╣т«џсЂЌсѓёсЂЎсЂЈсЂфсѓісЂЙсЂЎсђѓ

## ­ЪЊЮ сѓесЃЕсЃ╝ТЃЁта▒сЂ«ТДІжђа

```json
{
  "url": "https://example.com",
  "data": {
    "title": "Example Domain",
    "description": null,
    "price": "┬Ц1,980"
  },
  "errors": {
    "description": {
      "type": "error",
      "message": "УдЂу┤асЂїУдІсЂцсЂІсѓісЂЙсЂЏсѓЊсЂДсЂЌсЂЪ",
      "selector": {
        "type": "css",
        "value": "meta[name='description']",
        "transform": "attribute:content"
      }
    }
  }
}
```

## ­Ъћб сѓесЃЕсЃ╝сѓ┐сѓцсЃЌ

- `error`: жЄЇтцДсЂфсѓесЃЕсЃ╝сЂДсђЂУдЂу┤асЂїУдІсЂцсЂІсѓЅсЂфсЂёсЂІтЄдуљєсЂФтц▒ТЋЌсЂЌсЂЪта┤тљѕ
- `warning`: У╗йт║дсЂ«УГдтЉісЂДсђЂсѓфсЃЌсѓисЃДсЃісЃФсЂфУдЂу┤асЂїУдІсЂцсЂІсѓЅсЂфсЂёсЂІсђЂсЃЋсѓЕсЃ╝сЃФсЃљсЃЃсѓ»тђцсЂїСй┐ућесЂЋсѓїсЂЪта┤тљѕ

## ­ЪЊЮ сѓесЃЕсЃ╝сЃАсЃЃсѓ╗сЃ╝сѓИсЂ«СЙІ

- `УдЂу┤асЂїУдІсЂцсЂІсѓісЂЙсЂЏсѓЊсЂДсЂЌсЂЪ`: ТїЄт«џсЂЋсѓїсЂЪсѓ╗сЃгсѓ»сѓ┐сЂФСИђУЄ┤сЂЎсѓІУдЂу┤асЂїсЃџсЃ╝сѓИтєЁсЂФтГўтюесЂЌсЂфсЂё
- `тцЅТЈЏтЄдуљєсѓесЃЕсЃ╝`: УдЂу┤асЂ»УдІсЂцсЂІсЂБсЂЪсЂїсђЂТїЄт«џсЂЋсѓїсЂЪтцЅТЈЏтЄдуљє№╝ѕtext, html, attributeсЂфсЂЕ№╝ЅсЂ«жЂЕућесЂФтц▒ТЋЌсЂЌсЂЪ
- `сѓ╗сЃгсѓ»сѓ┐Тцюу┤бсѓесЃЕсЃ╝`: сѓ╗сЃгсѓ»сѓ┐сЂ«ТДІТќЄсЂїуёАті╣сЂДсЂѓсѓІсЂІсђЂТцюу┤бтЄдуљєСИГсЂФсѓесЃЕсЃ╝сЂїуЎ║ућЪсЂЌсЂЪ
- `сЃЄсЃ╝сѓ┐ТійтЄ║сѓесЃЕсЃ╝`: сЂЮсЂ«С╗ќсЂ«СИђУѕгуџёсЂфсѓесЃЕсЃ╝

## ­ЪЊЮ сѓесЃЕсЃ╝сЃЈсЃ│сЃЅсЃфсЃ│сѓ░сЂ«СЙІ

```python
from client import PlayScraperClient

client = PlayScraperClient()

# сѓ╣сѓ»сЃгсѓцсЃћсЃ│сѓ░сѓњт«ЪУАї
result = client.start_scraping("https://example.com", selectors={
    "title": "h1",
    "description": {
        "type": "css",
        "value": "meta[name='description']",
        "transform": "attribute:content"
    }
})

task_id = result["task_id"]
data = client.wait_for_completion(task_id)

# сѓесЃЕсЃ╝сЂ«уб║УфЇсЂетЄдуљє
if "errors" in data["result"]:
    for key, error in data["result"]["errors"].items():
        print(f"сѓесЃЕсЃ╝ ({key}): {error['message']}")
        print(f"сѓ╗сЃгсѓ»сѓ┐: {error['selector']}")
        print(f"сѓ┐сѓцсЃЌ: {error['type']}")
        print("---")

# сЃЄсЃ╝сѓ┐сЂ«тЄдуљє
print("ТійтЄ║сЂЋсѓїсЂЪсЃЄсЃ╝сѓ┐:")
for key, value in data["result"]["data"].items():
    print(f"{key}: {value}")
```
